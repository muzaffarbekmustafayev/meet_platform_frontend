# Zoom Clone - Frontend

This is the frontend client for the Zoom Clone application.

## 🛠 Tech Stack
- **React.js** (Vite)
- **Tailwind CSS**
- **Socket.io-client**
- **Simple-peer** (WebRTC)
- **Lucide React** (Icons)

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## 🔐 Role Features
- **Host**: Complete control, recording, co-host management.
- **Co-host**: Moderation powers (kick, mute, admit).
- **Participant**: Standard features, request to share.
- **Guest**: Waiting room only, no chat.

## 🖥 Screen Sharing
Includes a built-in audio mixer using Web Audio API to broadcast both microphone and system audio simultaneously.
